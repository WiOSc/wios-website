JAVA

  • Java

    Java is a general-purpose, class-based, object-oriented programming language designed for having lesser implementation dependencies. Usually credited to be the heart of the android operating system, its “write once, run anywhere” feature makes it a developer’s favorite. Java is often used to construct applications in laptops, data centres, game consoles.

    Objective

    • To provide essential skills for developing solutions in a structured, algorithmic format using pseudo code.
    • Introduction to implementation of the algorithms with computer programming and presentation of results in a clear manner.
    • To develop problem solving skills and core programming techniques.

    Outcomes

    • Students will learn how to express algorithms using sequencing, conditional and iterative constructs, and solve mathematical and scientific problems in Java language.


  • Object Oreinted Programming

    Object-oriented programming is a model that provides different types of concepts aiming to implement real-world entities in programs. inheritance, abstraction, polymorphism are just some of them.

    Objective

    • To understand the concepts of object-oriented, event driven, and concurrent programming paradigms and to develop skills in using these paradigms through Java.
    • To be aware of the important topics and principles of software development.
    • To learn how to write a computer program to solve specific problems.
    • To enable using the Java SDK environment to create, debug and run simple Java programs.

    Outcomes

    • Knowledge- Gaining knowledge of the structure and model of the Java programming language.
    • Understanding- Understanding the use of the Java programming language for various programming technologies.
    • Application- Learn how to develop software in the Java programming language.
    • Analysis- Evaluate user requirements for software functionality required to decide whether the Java programming language can meet user requirements.
    • Synthesis- Proposing the use of certain technologies by implementing them in the Java programming language to solve the given problem.
    • Evaluation- Developing an engineering approach to solving problems, starting from the acquired knowledge of programming and knowledge of operating systems.


  • Data Structures And Algorithms

    Data structures are the programmatic way of arranging data on a computer so that it can be accessed and updated efficiently. Linked list, Stack, Queue, and arrays are a few examples of data structures used in Java.

    Objective

    • Understanding basic concepts involving stacks, queues, lists, trees and graphs.
    • To understand the procedure of writing algorithms and develop a step by step approach in solving problems with the help of fundamental data structures,along with application of concepts such as searching and sorting techniques.

    Outcomes

    • Students will explore the theory of asymptotic notations,learn how to analyze simple algorithms and will understand how to design an efficient solution to a given problem using data analysis techniques.
